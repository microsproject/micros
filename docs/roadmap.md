# MicrOS Roadmap

## 0.1.0 (MVP)
- Cooperative scheduler
- Mutex, semaphore, message queue
- GPIO + UART drivers
- POSIX + nRF52840 DK ports

## 0.2.0
- Preemptive scheduling
- Tickless timers
- I2C + SPI drivers
- Logging system

## 0.3.0
- Simple FS integration (littlefs)
- Networking (UART/SLIP)

See GitHub milestones for up-to-date tracking.
